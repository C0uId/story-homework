答错时的体验设计

调研发现：答错不应该惩罚，而应该是"另一个学习机会"。关键是不让孩子产生挫败感。

答错时的故事呈现方式对比
A. 当前系统：Wrong Path 补充说明
"答错了？没关系！让我换个角度帮你理解——
3×4 其实就是 3 个 4 相加……（解释文字）"
问题：答错后变成"老师讲解"，打断了故事节奏，孩子没有继续看下去的动力
B. 分支叙事：答错走另一条路（推荐）
"小明算错了密码，大门没有打开。
'哎呀！' 一个声音从旁边传来。
是一只会说话的猫头鹰：
'年轻人，你算错啦。4+4+4 才等于 12 哦。
不过没关系——我从后面的小门带你进去吧。'
小明跟着猫头鹰穿过一条秘密通道……"
答错 = 遇到新角色 / 走隐藏路线 / 看到不同的剧情。错也有"奖励"
C. 勇气机制：答错不扣分，答对加倍
"小明算错了，但没关系！
师父说过：'每一次错误都是通往正确的阶梯。'
系统提示：+'探索值' 5分（答错也有奖励！）
现在，小明可以从头再试一次，
或者先听听猫头鹰的提示再作答。"
答错 = 获得"探索值" + 提示机会 + 重新尝试。鼓励大胆尝试
A

A. 分支叙事：答错走另一条路

答错后剧情走向不同的分支，遇到新角色、走隐藏路线。错误本身变成故事内容。

沉浸感强 · AI 生成需设计分支 Prompt
B

B. 勇气机制 + 二次机会（推荐）

答错获得"探索值"（不扣分），可以看提示后重试。两次都错才进入分支剧情。兼顾学习效果和体验。

最佳平衡 · 鼓励尝试 + 保证学习效果
C

C. A + B 结合

第一次答错 → 勇气机制（探索值+提示+重试）；第二次答错 → 分支叙事（走另一条路）。最完整的体验。

效果最好 · 但每题需生成两种分支
