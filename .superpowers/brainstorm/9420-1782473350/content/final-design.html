知识剧场 v2 - 完整设计方案

基于所有决策汇总，以下是完整的重设计方案。请审阅。

设计决策汇总
重构方向 方向 C：全面重构（故事 + 交互 + 系统）
首版范围 ① 故事引擎 + ② 交互重构（学习画像和社交留 v2）
叙事风格 幽默吐槽风为默认，设置中可切换悬疑/热血
知识嵌入 故事优先模型：纯故事幕 ⇄ 知识关卡幕交替
答错机制 勇气机制 + 二次机会 + 分支叙事兜底
模块一：故事引擎重构（Prompt + 结构）

1.1 新的 System Prompt 核心原则

  • 幽默优先：每幕至少一个笑点（吐槽、反转、拟人）
  • 角色个性：主角有鲜明性格（好奇、爱吐槽、偶尔犯傻）
  • 知识即钥匙：知识点是剧情推进的必要条件
  • 章节悬念：每章结尾必须有 cliffhanger

1.2 新的 Episode 结构

章节 = [纯故事幕A] → [知识关卡幕B] → [纯故事幕A] → [知识关卡幕B] → ...

纯故事幕：150-250字，推进剧情、发展角色、埋伏笔
知识关卡幕：80-120字铺垫 + 知识关卡 + 答对/答错分支

1.3 三种叙事风格 Prompt 模板

  • 幽默吐槽：大量对话、拟声词、角色吐槽、反转笑点
  • 悬疑解密：线索链、推理过程、真相揭示
  • 成长热血：挑战→失败→坚持→突破，情感驱动
模块二：交互重构（UI + 体验）

2.1 首页简化

  • 4步选择 → 2步（年级 → 学科），主题自动匹配
  • 新增"继续冒险"按钮（一键回到上次进度）
  • 新增每日推荐（基于学习数据推荐今天该学什么）

2.2 答题交互升级

  • 答对：✨ 粒子特效 + "太棒了！" + 探索值+10
  • 答错：💪 鼓励动效 + "再试试！" + 探索值+5 + 提示按钮
  • 连击奖励：连续答对3题触发"连击模式"（答对经验翻倍）
  • 倒计时可选（家长设置中开启/关闭）

2.3 角色系统微调

  • "境界"改名为更直观的"等级"（Lv.1-12）
  • 新增"探索值"概念（替代部分经验值，答错也有奖励）
  • 保留天赋、技能、性格探测系统
